# **Проект: Место**  
Проект создан в качестве задания от [**Яндекс Практикума**](https://praktikum.yandex.ru)

В ней используются:  
* JavaScript  
* Различные стили текстов  
* Ссылки  
* Flex  
* Grid  
* Iframe  
* Файловая структура сделана полностью по стандартам БЭМ.
